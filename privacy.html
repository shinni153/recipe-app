개인정보처리방침

최종 업데이트: 2026년 6월 13일

RecipeX(이하 "서비스")는 사용자의 개인정보를 소중히 여기며, 관련 법령에 따라 개인정보를 보호합니다. 본 방침은 서비스 이용 시 수집되는 정보와 그 활용 방법을 설명합니다.

1. 수집하는 정보

서비스 이용 시 아래 정보가 수집됩니다:

  • Google 로그인 정보 — 이름, 이메일 주소, 프로필 사진 (Google OAuth 제공)
  • 레시피 데이터 — 사용자가 추출 및 저장한 레시피 내용
  • 입력 데이터 — 레시피 추출을 위해 입력한 유튜브 URL 및 업로드한 음식 사진
  • 이용 기록 — 레시피 추출 및 저장 기록
  • 자동 수집 정보 — IP 주소, 브라우저 종류, 접속 기록, 기기 정보 (서비스 품질 개선 목적)

2. 정보 수집 목적

  • 회원 식별 및 로그인 서비스 제공
  • 레시피 추출 및 저장 서비스 제공
  • 서비스 개선 및 오류 분석
  • 사용자 문의 응대

3. 제3자 서비스 이용

서비스는 아래 제3자 서비스를 활용합니다:

  • Google OAuth — 로그인 인증
  • Supabase — 데이터 저장 및 인증 관리
  • Google Gemini AI — 영상 및 이미지 분석을 통한 레시피 추출
  • YouTube — 영상 자막 추출

⚠️ 사용자의 개인정보는 위 서비스 운영 목적 외에 제3자에게 판매하거나 제공하지 않습니다.

4. 정보 보관 및 삭제

  • 수집된 정보는 회원 탈퇴 시 즉시 삭제됩니다.
  • 앱 내 설정 → 회원 탈퇴를 통해 직접 삭제할 수 있습니다.
  • 이메일 요청 시 접수 후 영업일 기준 3일 이내 삭제 처리됩니다.
  • 삭제 요청: recipex.ai@gmail.com

5. 사용자 권리

사용자는 언제든지 아래 권리를 행사할 수 있습니다:

  • 본인의 개인정보 열람 및 수정 요청
  • 개인정보 삭제 요청 (앱 내 회원 탈퇴 또는 이메일 요청)
  • 개인정보 처리 정지 요청

6. 보안

서비스는 사용자 정보 보호를 위해 다음 조치를 취합니다:

  • HTTPS 암호화 통신
  • Supabase Row Level Security (RLS) 적용
  • Google OAuth 표준 인증 방식 사용

7. 아동 개인정보

서비스는 만 14세 미만 아동의 개인정보를 의도적으로 수집하지 않습니다. 만 14세 미만 사용자는 보호자의 동의 하에 서비스를 이용해 주세요.

8. 방침 변경

본 개인정보처리방침은 법령 변경 또는 서비스 업데이트에 따라 변경될 수 있습니다. 변경 시 앱 내 공지 또는 이메일을 통해 안내드립니다.
